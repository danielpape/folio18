Homestead A design system for editorial homepages.

Optimised for Adaptability, Hierarchy & Responsivity

Adaptability

There are 2 hierarchies for articles: hero and standard. The hero articles are intended for high impact stories and use an backgrounds and colour to elevate their visual level.

Heirarchy

There are 2 hierarchies for articles: hero and standard. The hero articles are intended for high impact stories and use an backgrounds and colour to elevate their visual level.

Responsivity

A page is made up of components which can be ordered in any way. Each component has options such as left or right alignment. The components are designed to have different information density so that the page rhythm can be set by the editor.

Fully Documented

All styles and design decisions documented with a Sketch file with symbols set up.
